To use:  run TracePenTestAll.bat inside of a command prompt with admin privileges
specify the passwords you want to search for inside of passwords.txt.
the resulting text containing those passwords, if any, can found in results.txt.

Make sure that some trace logs are already in C:\ProgramData\SSOProvider\Logs.


Documentation on other pen testing tools:
